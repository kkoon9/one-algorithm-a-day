## 설계

### 누적 합
누적 합 알고리즘을 통해 시간 초과를 피해야 한다.

N이 1024에 2차원배열이고, M이 100,000이므로 모든 구간 합을 구하려고 하면 시간초과가 발생한다.


## 시간

| 내 코드 (ms) | 빠른 코드 (ms) |
|:---------:|:----------:|
|    876    |      312      |

## 꼭 알아둬야 할 점
- 2차원 누적 합 알고리즘은 꼭 알아두자. S[secondX][secondY] - S[secondX][firstY] - S[firstX][secondY] + S[firstX][firstY]
