## 설계

### Deque
순서(order)와 중요도(importance)를 갖는 class(Document)를 타입으로 하는 Deque를 선언해준다.
 
원하는 순서의 문서가 나올 때까지 중요도 높은 문서를 차례로 제거해주며 카운트를 구한다.

시간 복잡도 O(N * M)
